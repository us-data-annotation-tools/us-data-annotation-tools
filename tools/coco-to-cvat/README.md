# COCO JSON to CVAT XML Converter

Part of the US-DATA Annotation Tools repository.

This tool converts COCO JSON datasets into CVAT XML annotation format for computer vision and machine learning workflows.

## Features

- Bounding Boxes
- Polygon Annotation
- Semantic Segmentation
- COCO JSON support
- CVAT XML export
- FastAPI server
- CLI support
